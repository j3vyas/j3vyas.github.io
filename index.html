Image Scraper

Python script that is able to download all images in a given url (and searches all sub-urls too)

This python script allows the user to specific the number of images to download, location of download, and url to download from. The script will download all images from the url provided, and then further dives deeper into the url to download all images within entire domain. The script utilizes multi-threading to be more efficient while waiting for I/O requests (HTTP GET requests and image downloads). Additionally, it is scheduled to time-out within 1 second of a request to avoid prolonging the run-time of the script (easily changable). Light-weight and extremely flexible script that is easy to pull, use, and modify.
